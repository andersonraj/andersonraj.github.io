# Project Overview
 
This is my very first website designed for all the car enthusiasts out there. It contains all the information that I have acquired from the web about all my favorite cars.


# Project Requirements

1. TextEdit(Mac)/NotePad(Windows)/VisualStudio Code(preferred)
2. Web-browser(Safari, Chrome, Firefox, Opera etc.)
3. Requires internet connection to load videos and images from external sources
4. If willing to use external Js sheet then javascript plugin is required for Visual Studio Code.
